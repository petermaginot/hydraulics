Classes
Line segment class
•	Profile variable: Consists of a list of ID, OD, wall thickness, distance-from-start, and elevation. Can alternatively be defined with hydraulic diameter D_h and flow_area in lieu of an ID or OD+wall thickness. Properties apply from each point forward in the segment.
•	Roughness variable (eps) : absolute roughness of the entire segment
•	Optional wall thermal conductivity variable (k) – Thermal conductivity pipe wall of the entire segment – nothing is done with this variable now, but later support will be added for heat transfer using this variable.
•	Has dP function to calculate differential pressure from friction and elevation change across this segment given an Incompressible_Fluid object and a flow rate. Steps through segment and calculates the friction and expansion/contraction pressure drop at each segment, as well as the pressure drop due to acceleration/deceleration with the Bernoulli equation. Negative means pressure at end of slice is lower than pressure at beginning.
    For each slice:
    dP_slice = dP due to friction + dP due to elevation change + (dP due to expansion/contraction and velocity change from previous segment if not the first segment).
    Note: If noncircular (i.e. if defined with D_h and flow_area instead of Di) then ignore dP due to expansion/contraction)
    dP = sum of dP_slice over all slices
Bend class
•	Internal diameter variable (Di)
•	Bend angle variable (ang_deg): bend angle, in degrees
•	Bend diameters variable (bend_dias) : Bend diameter in multiples if internal diameter (i.e. standard long-radius 1.5 * Di bend : bend_dias = 1.5)
•	dP function that calls dP_bend
Contraction/expansion class
•	Upstream diameter variable (Di_US)
•	Downstream diameter variable (Di_DS)
•	dP function that calls dP_contraction_expansion

Incompressible_Fluid class
•	Density variable (rho)
•	Viscosity variable (mu)

Global functions
dP_friction
•	Calculates permanent friction pressure drop for a given length slice

dP_bend
•	Calculates permanent friction pressure drop for a pipe bend

dP_contraction_expansion
•	Calculates permanent friction pressure drop for an expansion or contraction
